Introduction

Cette page donne quelques informations sur l’utilisation du manuel

Utiliser le manuel

Ouverture

man rm
man cp
man mv

Fermeture

Appuyer sur la touche q pour quitter

Navigation

Utiliser les touches suivantes:

  • flèches ↑ et ↓
  • touches u, d, j ou k

Présentation d’une page typique

Paragraphe NAME

Donne une description succincte de ce que permet de réaliser la commande

Paragraphe SYNOPSYS

Donne un aperçu de la manière d’utiliser la commande.

Une notation spécifique est utilisée pour représenter les paramètres et options:

commande EXPRESSION        EXPRESSION est obligatoire
commande [EXPRESSION]      EXPRESSION est facultatif
commande EXPRESSION...     EXPRESSION est obligatoire et peut être multiple
commande [EXPRESSION]...   EXPRESSION est facultatif et peut être multiple

Donne une description précise des options/usages

Les raccourcis utilisables

Ce sont les mêmes que ceux de la commande less

Commande less

La commande less permet en plus de naviguer dans le fichier. Les touches usuelles dans cette commande sont:

  • q : quitter
  • u : remonter d’une page
  • d : descendre d’une page
  • flèches haut/bas: monter/descendre
  • g : aller au début du fichier
  • G (⇧g): aller à la fin du fichier
  • / : chercher une chaîne (taper ensuite la chaîne à rechercher puis ↵)
    • n : aller à l’occurrence de recherche suivante
    • N (⇧n): aller à l’occurrence de recherche précédente

EXEMPLE

Exécuter la commande less /etc/passwd

  • Presser /
  • Saisir bash
  • Presser ↵ (une première occurence est trouvée)
  • Presser n (l’occurrence suivante est sélectionnée)
  • Presser ⇧n (l’occurrence précédente est sélectionnée)
  • -I puis ↵: respecte la casse dans les recherche
  • -i puis ↵: ignore la casse dans les recherche
  • -OPTION puis ↵: active l’option OPTION désignée par une lettre (voir man less)
